# Array vs Linked List

| 		Parameter			| Array		    | Linked List	| Better for parameter	|
| ------------- 	| ------------- | ------------- | ------------- |
| Cost of accesing  | Constant time | Average case 	| Array			|
| an element  		|     O(1)  	| 	 	O(n)	|				|
|					|				|				|				|
| ------------- 	| ------------- | ------------- | ------------- |
| Memory requirements | Fixed size	| - No unused memory | 			| 
|					|				| - Extra memory for pointer variables		|				|
